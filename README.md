## Features

- TypeScript
- React
- Redux
- Bootstrap
- Sass
- React CSS Modules (CSS Modules)
- **HOT RELOAD**

## Usage 

[Download](https://github.com/praneshr/smart-TSX-boilerplate/archive/master.zip) the latest zip file.

## Development Setup
```bash
yarn install
yarn run build
yarn start
```
